HTML Basics

HTML (HyperText Markup Language) is the foundation of web development.

<!DOCTYPE html>
<html>
    <head>
        <title>My First Page</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
</html>

CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements.

body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
}

h1 {
    color: darkblue;
    text-align: center;
}

Practice

Try editing the HTML and CSS above in a code editor like VS Code or an online editor like CodePen.
